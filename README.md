# Portfolio REST API

Här är REST-webbtjänst med data för olika portfolioinnehåll.

* Fullt stöd för CRUD med GET POST PUT DELETE
* Olika data kan konsumeras från olika endpoints

    - /completedstudies
    - /workexperience
    - /webpages
